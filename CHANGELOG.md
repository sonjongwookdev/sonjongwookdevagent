### 추가됨

- 💬 AI 채팅 인터페이스 (웹뷰)
- 🔧 코드 리팩토링 기능
- 🐛 자동 버그 수정 기능

- 💡 코드 설명 기능
- 🎯 컨텍스트 인식 (현재 파일, 에러, 선택 영역)

- 🎨 스트리밍 응답 지원
- ⌨️ 키보드 단축키 지원
